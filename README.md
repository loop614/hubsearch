## Description
- example symfony project
- search Github issues and display number of rocks/sucks words with 1-10 grade
- has redis cache

## Requirements
- make
- docker with compose

## Quick start
- make init
- open localhost:12345/score on your favourite browser

## Run tests
- make test_init
- make test
